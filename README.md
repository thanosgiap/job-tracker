# Job Tracker

A full-stack job application tracker built with Next.js, TypeScript, and AI-powered auto-fill.
You can view it here [job-tracker](https://job-tracker-mu-two.vercel.app)

## Features

- **Kanban board** — drag and drop applications between status columns
- **AI auto-fill** — paste a job URL and the form fills itself using Claude AI
- **Table view** — sortable list view of all applications
- **Stats dashboard** — track your interview rate, offer rate and weekly activity
- **CSV export** — download all your applications as a spreadsheet
- **Authentication** — secure login with Clerk

## Tech stack

- **Framework** — Next.js 15 (App Router)
- **Language** — TypeScript
- **Styling** — Tailwind CSS + shadcn/ui
- **Database** — PostgreSQL via Supabase
- **ORM** — Prisma
- **Auth** — Clerk
- **AI** — Anthropic Claude API
- **Drag and drop** — dnd-kit
- **Data fetching** — React Query
- **Forms** — React Hook Form + Zod
- **Charts** — Recharts
- **Deployment** — Vercel
